Introdução ao MVC

O padrão MVC(model-view-controller)é uma arquitetura de software utilizada no desenvolvimento de aplicações

para separar a lógica de negócios,a interface do usuário e o controle dos dados.

Componentes do MVC

  • Model(Modelo) - Representa os dados e a lógica de negócio da aplicação.
  • View(Visão) - Responsável pela interface com usuário e exibição dos dados.
  • Controller(Controlador) - Intermedia a comunicação entre model e view.
